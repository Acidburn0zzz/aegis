Aegis

To start using the Aegis UI, download the compiled css and javascript files. Include them in your project, and start designing. The only requirement for Aegis is jQuery, which is required for animations and rendering advanced form elements.

So, what are you waiting for? Click the shiny green button and get going! You can also have a look at the source files if you want to customize or contribute to the project.

Usage

To use Aegis in your project, link to the css and js files as such: 

<link rel="stylesheet" href="<path-to-css>/aegis.css">
<script type=&#039;text/javascript&#039; src=&#039;<path-to-js>/jquery.js&#039;></script>
<script type=&#039;text/javascript&#039; src=&#039;<path-to-js>/aegis.js&#039;></script>
